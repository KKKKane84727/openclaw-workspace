# Lyrics Composition Guide

Phase 2 作词参考。Cartooner 根据风格/主题创作歌词时遵循以下规范。

## 结构标签

MiniMax 支持 14 种官方歌词结构标签，模型会据此理解歌曲段落并分配编曲：

| 标签 | 用途 | 编曲暗示 |
|------|------|---------|
| `[Intro]` | 前奏 | 乐器渐入，氛围铺垫 |
| `[Verse]` / `[Verse 1]` | 主歌 | 叙事段，配器相对简洁 |
| `[Pre Chorus]` | 预副歌 | 情绪递进，过渡到高潮 |
| `[Chorus]` | 副歌 | 旋律最强，配器最饱满 |
| `[Post Chorus]` | 副歌后段 | 副歌余韵，hook 重复 |
| `[Hook]` | 记忆点 | 最抓耳的短乐句 |
| `[Bridge]` | 桥段 | 打破模式，新鲜感 |
| `[Drop]` | 落拍（电子乐） | 节奏释放，低频爆发 |
| `[Build Up]` | 铺垫（电子乐） | 能量持续累积 |
| `[Solo]` | 乐器独奏 | 吉他/钢琴/萨克斯 solo |
| `[Inst]` | 纯器乐段 | 无人声的间奏 |
| `[Interlude]` | 间奏 | 段落间的过渡 |
| `[Break]` | 停顿/留白 | 突然安静，制造对比 |
| `[Transition]` | 过渡 | 风格/速度转换 |
| `[Outro]` | 尾奏 | 渐弱或突然结束 |

> **标签越精确，编曲越智能。** 不写标签的歌词 = 一整块文本，模型无法区分段落功能。

## 标准歌曲结构模板

```
[Intro]
(描述前奏氛围，如：轻柔钢琴引入，4小节)

[Verse 1]
每行 5-12 个字（中文）或 6-12 个音节（英文）
保持每行节奏一致
本段 4-6 行

[Pre Chorus]
过渡段，2-4 行
情绪递进，引向高潮

[Chorus]
最记忆深刻的部分
旋律感最强，重复性高
4-6 行，行长与 Verse 接近

[Verse 2]
延续 Verse 1 的韵律模式
但内容推进故事/情感

[Chorus]
（重复 Chorus，可微调 1-2 行）

[Bridge]
转折段，2-4 行
打破前面的模式，带来新鲜感

[Chorus]
（最终 Chorus，可加强）

[Outro]
(描述尾奏，如：渐弱淡出)
```

## 电子乐结构模板

```
[Intro]
[Build Up]
[Drop]
[Break]
[Verse]
[Build Up]
[Drop]
[Bridge]
[Drop]
[Outro]
```

## Quick Template (30s 短歌)

```
[Verse]
（4 行，叙事/场景）

[Chorus]
（4 行，情感高潮/hook）
```

## Full Template (60-120s)

```
[Intro]
（氛围描述）

[Verse 1]
（4-6 行）

[Chorus]
（4-6 行）

[Verse 2]
（4-6 行）

[Chorus]
（重复/变化）

[Bridge]
（2-4 行）

[Chorus]
（最终）

[Outro]
（淡出描述）
```

## 写作规范

**韵脚**（默认必须，按风格豁免）：
- **流行/民谣/抒情/爵士/R&B**：尾韵必须。中文押韵母（ang/ang、ou/ou、i/i 等），英文 AABB 或 ABAB。每个段落（Verse/Chorus）内至少隔行押韵
- **说唱/Hip-Hop**：内韵（行内押韵）优先于尾韵，允许半韵和近韵
- **实验/后摇/意识流**：意象和情感优先，韵脚降级为加分项
- **其他风格**：默认要求尾韵
- **跨段韵脚一致性**：Chorus 的韵脚应在重复时保持一致；V1 和 V2 建议使用相同韵部

**避免**: 空洞口号、过度堆砌形容词、陈词滥调（"追逐梦想"、"星辰大海"）。不要为了凑韵脚而使用陈词——宁可换韵部也不要降低意象质量。
**好的歌词**: 有画面感、有具体意象、情感真实不矫揉。

**写完歌词后，向用户展示并确认**，再进入 Phase 3。
